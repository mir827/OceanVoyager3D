# Ocean Voyager 3D

대항해시대의 미지의 바다를 탐험하는 브라우저 기반 3D 어드벤처 게임입니다.

## Play

- `W` / `↑`: 전진
- `S` / `↓`: 감속 및 후진
- `A` / `←`, `D` / `→`: 방향 전환
- `Space`: 대포 발사
- 모바일에서는 화면 버튼을 사용합니다.

다섯 섬의 보물을 모으고 폭풍과 해적선을 피해 왕실 항구로 돌아오세요.

## Development

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```
